Creating & Generating API keys from your WooCommerce Website

Before initiating to use the module you need to integrate the connector via API keys for that you need to generate the API from Woocommerce following are the list of first steps towards the integration. In your Woocommerce backend you need to go into Woocommerce > Settings > and then click on API as showed in the figure.

Once you click on the API button you need to “Enable the Rest API” once you enable you can click on Save Button.

You need to create the API key which will be interacting with Odoo ERP software for that you can go into settings Click on Add keys – A pop up will open where you need to put the user name and provide permissions. Kindly document these keys well in your email or any folder.
